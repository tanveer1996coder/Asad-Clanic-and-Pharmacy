# Medical Store Audit - Keyboard Shortcuts & Commands Guide

## Global Shortcuts (Available Everywhere)

| Shortcut | Action |
|----------|--------|
| `Shift + ?` | Show Keyboard Shortcuts Help Modal |
| `Alt + 1` | Navigate to Dashboard |
| `Alt + 2` | Navigate to Sales Page |
| `Alt + 3` | Navigate to Products Page |
| `Alt + 4` | Navigate to Stock Page |
| `Alt + 5` | Navigate to Customers Page |
| `Alt + 6` | Navigate to Suppliers Page |
| `Alt + 7` | Navigate to Purchase Orders Page |

---

## Sales Page (`/sales`)

### Product Search & Selection
| Shortcut | Action |
|----------|--------|
| `F2` | Focus on Product Search Bar |
| `Escape` | Clear selection / Blur search field |
| `Enter` | (In Search) Add selected product to cart |

### Customer Selection
| Shortcut | Action |
|----------|--------|
| `F3` | Focus on Customer Field |
| Type new name | Shows "➕ Add [Name]" option to create new customer |

### Cart Actions
| Shortcut | Action |
|----------|--------|
| `F8` | Toggle Unit (Box/Item) of last added item |
| `F9` | Focus on Discount Field |

### Checkout
| Shortcut | Action |
|----------|--------|
| `F4` | Complete Sale / Checkout |

---

## Sales History Page (`/sales-history`)

### Tabs
- **Invoices Tab**: View complete transaction invoices
- **Item Sales Tab**: View individual items sold

### Actions
| Action | Description |
|--------|-------------|
| Click Reorder Icon (↻) | Load all items from a past invoice into Sales cart for re-ordering |

## Keyboard Navigation Tips

1. **Forms**: Press `Enter` to move between fields quickly (works in Product and PO forms)
2. **Customer Creation**: Type a new name in Sales → Customer field to instantly create new customers
3. **Mobile**: Forms automatically adapt to card layout on small screens
4. **F-Keys**: Function keys (F2-F9) are optimized for quick sales workflow

---

## Mobile Responsiveness

- **Product Forms**: Fields stack vertically on mobile
- **Purchase Order Items**: Display as individual cards instead of table on mobile
- **Tables**: Horizontal scroll enabled for all data tables

## Feature-Specific Commands

### Customer Management (Sales Page)
1. Focus customer field (`F3`)
2. Type new customer name
3. Click "➕ Add [Name]" or press `Enter`
4. Fill optional details (phone, address)
5. Customer auto-selected for current sale

### Receipt Reordering (Sales History)
1. Go to Sales History
2. Switch to "Invoices" tab
3. Find invoice
4. Click Reorder button (↻)
5. Edit prices/quantities in Sales cart
6. Complete new sale (`F4`)

### Box/Item Toggle (Sales)
- Add product to cart
- Press `F8` to switch between selling by Box or Item
- Only works on last added item
